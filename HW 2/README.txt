Procedural overhead cable.

Steps:

1. Scatter points on a surface
2. Copy towers at each point
3. Scatter beams on each tower
4. Add a bunch of points to each beam representing cable connections
5. For each cable connector, make a cable (polyline) from its point to a random cable connector that isn't on the same tower
6. Droop cables (https://youtu.be/PTVal-t9g1k?t=1060)
7. Polywire each cable for thickness
8. Profit
